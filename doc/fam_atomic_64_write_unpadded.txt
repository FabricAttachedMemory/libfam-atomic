fam_atomic_64_write_unpadded(3)
===============================
:doctype: manpage

NAME
----
fam_atomic_64_write_unpadded - unpadded 64 bit fam atomic write

SYNOPSIS
--------
....
#include <fam_atomic.h>

void
fam_atomic_64_write_unpadded(int64_t *address,
			     int64_t value);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 64 bit fam
atomic variable, sets the atomic variable to 'value'. This is
essentially an atomic swap that does not return a value.

RETURN VALUE
------------
None
