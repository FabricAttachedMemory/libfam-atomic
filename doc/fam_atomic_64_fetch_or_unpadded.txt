fam_atomic_64_fetch_or_unpadded(3)
===================================
:doctype: manpage

NAME
----
fam_atomic_64_fetch_or_unpadded - unpadded 64 bit fam atomic fetch or operation

SYNOPSIS
--------
....
#include <fam_atomic.h>

int64_t
fam_atomic_64_fetch_or_unpadded(int64_t *address,
                                int64_t arg);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 64 bit fam
atomic variable, reads the contents of the atomic variable,
performs bitwise OR between the atomic variable and 'arg',
and returns the original value that was read.

RETURN VALUE
------------
Returns the previous value of the fam atomic (before the atomic
variable was OR'd with 'arg'.
