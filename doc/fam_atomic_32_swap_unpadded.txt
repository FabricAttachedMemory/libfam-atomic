fam_atomic_32_swap_unpadded(3)
==============================
:doctype: manpage

NAME
----
        fam_atomic_32_swap_unpadded - unpadded 32 bit fam atomic swap

SYNOPSIS
--------
        #include <fam_atomic.h>

        int32_t
        fam_atomic_32_swap_unpadded(int32_t *address,
                                    int32_t value);

        cc ... -lfam_atomic

DESCRIPTION
-----------
        Given 'address', which is a pointer to an unpadded 32 bit fam
        atomic variable, reads the contents of the atomic variable,
        sets the variable to 'value', and returns the original
        value that was read.

RETURN VALUE
------------
        Returns the previous value of the fam atomic (before it
        was set to 'value').
