fam_atomic_128_write_unpadded(3)
================================
:doctype: manpage

NAME
----
fam_atomic_128_write_unpadded - unpadded 128 bit fam atomic write

SYNOPSIS
--------
....
#include <fam_atomic.h>

void
fam_atomic_128_write_unpadded(int64_t *address,
			      int64_t value[2]);

cc ... -lfam_atomic
....
DESCRIPTION
-----------
Given 'address', which is a pointer to an unpadded 128 bit fam
atomic variable, sets the atomic variable to 'value'. This is
essentially an atomic swap that does not return a value.

'value' is an array of two 64 bit integers representing
a 128 bit variable, where index [0] represents the first
64 bits and index [1] represents the last 64 bits.

RETURN VALUE
------------
None
